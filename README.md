Wordle Game

A browser-based clone of the popular word-guessing game Wordle. Guess the hidden 5-letter word in 6 tries or fewer!

🎮 Features


🟩 Classic Wordle gameplay — guess a 5-letter word within 6 attempts
🎨 Color-coded feedback:

Green — correct letter, correct position
Yellow — correct letter, wrong position
Gray — letter not in the word



⌨️ On-screen keyboard with live letter status
📱 Responsive design for desktop and mobile
🔁 New random word each game
📊 (Optional) Win/loss streak tracking


🕹️ How to Play


You have 6 attempts to guess the secret 5-letter word.
Type a valid word and press Enter to submit your guess.
After each guess, the tiles change color to show how close you were:

Green = right letter, right spot
Yellow = right letter, wrong spot
Gray = letter isn't in the word at all



Use the feedback to narrow down your next guess.
Guess the word correctly before you run out of attempts to win!
